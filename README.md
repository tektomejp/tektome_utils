# openflow_schema
OpenFlow schema via pydantic classes

## Installation

You can install this package directly from GitHub:

```bash
pip install git+https://github.com/tektomejp/openflow_schema.git@main
```

## Usage

```python
import openflow_schema

# Your OpenFlow schema code here
```

## Development

To install in development mode:

```bash
git clone https://github.com/tektomejp/openflow_schema.git
cd openflow_schema
pip install -e .
```
